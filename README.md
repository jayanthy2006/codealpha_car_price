# Car Price Prediction using Machine Learning

## Project Overview
This project predicts the selling price of a car using Machine Learning techniques in Python.  
The model is trained using Linear Regression on the car dataset.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Dataset
Dataset used:
- car data.csv

The dataset contains:
- Car Name
- Year
- Present Price
- Kms Driven
- Fuel Type
- Selling Type
- Transmission
- Owner
- Selling Price

---

## Machine Learning Algorithm
- Linear Regression

---

## Features of Project
- Data preprocessing
- Label Encoding
- Train/Test Split
- Model Training
- Price Prediction
- Accuracy Evaluation
- Graph Visualization
